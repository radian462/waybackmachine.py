# waybacktools
Unofficial Python API Wrapper for Wayback Machine.  

# Features
## **No API Key Required**
This library does not require an API key.

## **Resource Retrieval**
You can retrieve resources from archives at the time of saving.

## **Download Functionality**
Internally uses Playwright to download archives in MHTML or PDF format.

# Installation
```bash
pip install waybacktools
playwright install
```

# Usage
## Save
```python
from pprint import pprint
from waybacktools import waybacktools

wayback = waybacktools()
pprint(wayback.save("https://www.google.co.jp/"), sort_dicts=False)
```
