# waybacktools
waybackmachineのpythonの非公式APIラッパー  
現在開発中

# 特徴
## **API Keyが不要**
このライブラリにはAPI Keyが必要ありません。

## **リソースの取得**
保存時にアーカイブのリソースを取得できます。

## **ダウンロード機能**
内部でPlaywrightを利用し、MHTML形式またはPDF形式でアーカイブをダウンロードできます。

# 導入
```bash
pip install waybacktools
playwright install
```
